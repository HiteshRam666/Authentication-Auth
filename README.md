# FastAPI JWT Authentication Demo

This project demonstrates how to implement **OAuth2 Password Flow with JWT (JSON Web Tokens)** using **FastAPI**. It includes secure password hashing, token generation, token validation, and protected routes.

This implementation is ideal for learning purposes, demos, and as a base for building production-ready authentication systems.

---

## Features

- OAuth2 Password Grant authentication
- JWT access token generation and validation
- Secure password hashing using SHA-256 + bcrypt
- Protected routes using FastAPI dependencies
- Pydantic models for request/response validation
- Interactive API documentation (Swagger UI)
